# **App Name**: GlamGo

## Core Features:

- Artist Profile Display: Display profiles of mehndi and makeup artists, including work photos, service types (mehndi, makeup, or both), location, and charges.
- Service Filtering: Allow users to filter artists by location, price, service type (mehndi or makeup), and availability. User interface elements should allow searching using location, city, or pin code.
- Personalized Recommendations: Suggest Mehndi and Makeup artists to users using an AI tool that suggests items tailored to their history and style preferences.
- Booking Request: Allow customers to enter service date/time, and address.
- Artist Locator: Display nearby artists based on customer location

## Style Guidelines:

- Primary color: Rich Henna (#8B4513) to represent tradition and artistry.
- Background color: Soft Sand (#F5F5DC) for a warm and inviting feel.
- Accent color: Golden Bronze (#CD7F32) to highlight key actions and elements, adding a touch of sophistication and glamour.
- Secondary color: Light Pink (#FFB6C1) to evoke femininity and grace.
- Body and headline font: 'Mrs Saint Delafield' for headings to convey elegance, paired with 'Lato' for body text for readability and a modern touch.
- Use refined and minimalist line icons in golden bronze that represent beauty, makeup tools, and mehndi patterns.
- Clean and spacious layout with a focus on high-quality images of artists' work, ensuring a visually appealing and easy-to-navigate experience.
- Subtle transitions and animations to enhance user engagement and provide feedback, such as smooth loading indicators and interactive elements.
